## 8. Accessibility

### 8.1 WCAG 2.1 Guidelines

| Guideline | Requirement | Implementation |
|-----------|-------------|----------------|
| Color Contrast | 4.5:1 for text | Verified with contrast checker |
| Keyboard Navigation | All interactive elements | tabIndex, focus states |
| Screen Readers | Proper ARIA labels | aria-label, role attributes |
| Focus Indicators | Visible focus state | ring-2 ring-offset-2 |
| Error Messages | Clear, descriptive | Below input, aria-describedby |

### 8.2 Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Tab` | Move to next focusable element |
| `Shift + Tab` | Move to previous focusable element |
| `Enter` | Activate button/link |
| `Escape` | Close modal/dropdown |
| `Arrow keys` | Navigate within dropdown/menu |
| `/` | Focus search input (future) |

## 10. Animation & Motion

### 10.1 Animation Guidelines

| Animation | Duration | Easing | Usage |
|-----------|----------|--------|-------|
| Hover | 150ms | ease-out | Buttons, cards |
| Focus | 150ms | ease-out | Form elements |
| Modal open | 200ms | ease-out | Modals, dialogs |
| Modal close | 150ms | ease-in | Modals, dialogs |
| Page transition | 300ms | ease-in-out | Page changes |
| Loading pulse | 2s | ease-in-out | Skeletons |

### 10.2 Tailwind Animation Classes

```css
/* Fade in */
.animate-fade-in {
  animation: fadeIn 200ms ease-out;
}

/* Slide up */
.animate-slide-up {
  animation: slideUp 200ms ease-out;
}

/* Pulse (loading) */
.animate-pulse {
  animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
}

/* Spin (loading spinner) */
.animate-spin {
  animation: spin 1s linear infinite;
}
```

### 10.3 Motion Preferences

```typescript
// Respect user's reduced motion preference
const prefersReducedMotion = window.matchMedia(
  '(prefers-reduced-motion: reduce)'
).matches;

// Disable animations if user prefers reduced motion
if (prefersReducedMotion) {
  document.documentElement.classList.add('reduce-motion');
}
```
